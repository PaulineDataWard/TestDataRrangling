# Create a remote repo
On github.com: 
* Click 'Create a new repo' foo-repo
In git bash:
mkdir ~/foo-repo
cd ~/foo-repo
# Initialise local git repo
git init
# Point the local repo to the new remote repo just created on github.com
# NOT git remote add origin https://github.com/paulinedataward/training.git
git remote set-url origin https://github.com/paulinedataward/training.git

# Forking Fabulous Repos
On github.com, find a repo, click on 'Fork'
In git bash: 
git clone https://github.com/PaulineDataWard/MLNotes.git
